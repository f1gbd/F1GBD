Après le flashage

Au démarrage : logo MeshCore (~3 s) puis la page d'accueil pager (« RASEC ALERT / by F1GBD / Alertes: 0 »). Réveiller l'écran avec le bouton USER si besoin.

Pour déclencher une alerte, envoyer depuis TCQ ou une appli cliente MeshCore (message direct ou canal) :

#ra ADRASEC77

Le pager affiche « RASEC ALERT », la LED clignote par séries de 3 impulsions, le buzzer bipe (si équipé), et l'émetteur reçoit un accusé de réception (message direct).
