Saitama T-Deck — option RASEC-ALERT

Firmware pour LilyGo T-Deck / T-Deck Plus (ESP32-S3, LoRa SX1262), version …. Flashez directement depuis ce navigateur, sans rien installer.

Pour flasher :

  1. Utilisez Chrome ou Edge sur ordinateur (Web Serial requis).
  2. Branchez le T-Deck en USB-C, interrupteur latéral sur ON.
  3. Cliquez sur Installer ci-dessous et choisissez le port du T-Deck.
  4. Laissez l'installation se terminer, puis appuyez sur reset.

Navigateur non compatible — utilisez Chrome ou Edge sur ordinateur. Accès au port série refusé.

Si le bouton ne trouve pas le port ou échoue à se connecter (fréquent sur ESP32-S3 à USB natif) : mettez le T-Deck en mode download à la main — maintenez le bouton central de la trackball, appuyez sur reset (côté gauche), relâchez les deux — puis recliquez sur Installer.

Après le flash : déposez redalert.mp3 à la racine de la carte SD (FAT32) pour activer le son d'alerte, puis testez en envoyant #ra ADRASEC77 depuis un autre nœud MeshCore, en direct ou sur le canal privé partagé.

Dérivé de Saitama (GPL-3.0-or-later), basé sur MeshCore. Portage T-Deck + option RASEC-ALERT : F1GBD — ADRASEC 77.
