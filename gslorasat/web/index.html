Flashage One-Click de la station

Branchez votre carte en USB et cliquez sur « Installer » — le firmware est envoyé directement depuis votre navigateur. Aucun logiciel à installer.

La puce est détectée automatiquement : le bon firmware (T-Beam ou Heltec) est choisi tout seul.

Cartes supportées
CartePuce détectéeFirmware flashé
LilyGo T-Beam v1.1 (433 MHz, GPS)ESP32gslorasat-firmware-tbeam-433
Heltec WiFi LoRa 32 V3ESP32-S3gslorasat-firmware-heltec-v3
Prérequis
  1. Navigateur Chrome ou Edge sur ordinateur (le flashage web utilise le Web Serial ; Firefox et Safari ne le supportent pas).
  2. Le pilote USB-série de la carte installé (CP210x pour Heltec, CH9102 pour T-Beam) — en général déjà présent sous Windows 11.
  3. Fermez tout logiciel qui occupe le port (moniteur série, l'app GSLoRaSat…).
Ensuite
  1. Récupérez l'application GSLoRaSat.exe dans les Releases.
  2. Branchez la station, ouvrez l'app, sélectionnez le port USB, Connecter.
  3. Menu Éphémérides → Mettre à jour, puis Satellites → Mode automatique.

Flashage manuel possible via esptool : esptool --chip esp32 --port COMx write_flash 0x0 gslorasat-firmware-<carte>.bin
