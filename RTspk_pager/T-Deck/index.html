Compatible Chrome / Edge sur ordinateur (Web Serial). Ne fonctionne pas sur iOS / Safari.

Le bouton ne se connecte pas ? (fréquent sur ESP32-S3 à USB natif) Passez le T-Deck en mode download : maintenez la trackball enfoncée, appuyez brièvement sur reset (côté gauche), relâchez — puis recliquez sur « Installer ». Après le flash, appuyez sur reset.

▸ Étapes

  1. Ouvrir cette page depuis Chrome ou Edge sur ordinateur.
  2. Brancher le T-Deck en USB-C, interrupteur sur ON.
  3. Cliquer Installer le firmware et choisir le port série du T-Deck.
  4. Laisser l'installation se terminer, puis appuyer sur reset.
  5. Au 1er démarrage, régler la région radio sur Europe (868 MHz) dans Settings → Radio.

▸ Utilisation de l'alerte

Depuis un autre nœud Reticulum/LXMF, en message direct vers l'adresse LXMF du T-Deck :

CommandeEffet
#ra ADRASEC77Déclenche l'alerte : écran plein écran clignotant « RASEC ALERT » + sirène + accusé de réception.
#rapass <ancien> <nouveau>Change le code d'activation à distance (persisté en flash).
#b <n>Nombre de répétitions de la sirène. #b 0 = alarme continue jusqu'à acquittement. Plage 0–20.

Acquittement : toucher l'écran, appuyer sur une touche, ou appui long. La sirène est synthétisée dans le firmware (aucune carte SD ni MP3) et suit le volume / l'interrupteur audio.
